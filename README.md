## i18n

It is a french résumé.

## data

Data in the website are provided by `src/assets/json/data.json` file.

## usage

Install dev dependencies
`npm install`

Install project dependencies
`bower install`

Run in dev mode
`gulp serve`

Test
`gulp test`

Build
`gulp build`
