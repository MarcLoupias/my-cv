Proof of Concept Play!-AngularJs

Le projet montre une architecture possible pour un projet d'application web avec le Play framework et le duo Angular/Bootstrap.

    Le projet est scindé en deux :
  • La partie back contient la base de données, le serveur d'application et sa façade RESTful.
  • La partie front attaque les services REST du back pour nourrir le site.

Le projet est hébergé sur heroku, le back en tant qu'application Play!, le front en tant qu'application node.js

Technologies

Heroku, MySQL, Play!, AngularJs, Bootstrap

Langages

SQL, jPQL, Java, HTML5, CSS, Javascript
